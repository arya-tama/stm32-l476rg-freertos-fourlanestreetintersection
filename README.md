# stm32-l476rg-freertos-fourlanestreetintersection

Implementation of https://github.com/arya-tama/esp32_bare_metal_4_lane_street_intersection using STM32 Nucleo Board. 
Changes made: 
- 1x ESP32 WROVER -> 1x STM32 Nucleo-64 board with STM32L476RG MCU
- 1x Freenove 8 RGB LED Module -> 4x traffic light LED module
- 2x 74HC595 IC & 8x Resistor 220Ω & 1x 8x8 LED matrix -> 1 x 8x8 Led Matrix Module MAX7219
